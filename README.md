BigPipe原理
============================
BigPipe原理即根据页面内容位置的不同，将整个页面分成不同的块儿，分块输出，简单说就像一条流水线一样。这样就做到了浏览器和服务器的并行化，从而达到重叠服务器端运行时间和浏览器端运行时间的目的。
